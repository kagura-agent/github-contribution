# Weekly Contribution Reflections

Per-repo retrospectives on open-source contribution quality and outcomes.

Generated every Sunday 20:00 CST by `contribution-reflect` cron job.

## Format

Each file: `YYYY-MM-DD.md` (Sunday date)

Per repo section includes:
- PRs submitted / merged / rejected / open
- Maintainer feedback summary
- Merge rate
- Decision: continue / pause / blocklist
